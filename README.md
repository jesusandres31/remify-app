## Remify App 👨‍💼💼

SaaS webapp for business management.
Available in three languages (english, spanish and portuguese)
  

## Stack used 💻

- Node
- Express
- Typescript
- PostgreSQL
- React
- Redux-Toolkit
- Material-UI
- Docker
- Docker-Compose
- Nginx
